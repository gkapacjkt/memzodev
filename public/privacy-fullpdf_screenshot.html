Privacy Policy

Extension: Text Print & Full Page Screenshot
Last updated: June 25, 2026

1. Overview

Text Print & Full Page Screenshot ("the Extension") is a Chrome browser extension that captures web pages and selected text as PDF files. We are committed to protecting your privacy. This policy explains what data the Extension accesses, how it is used, and what is never collected.

2. Data We Do NOT Collect

The Extension does not collect, transmit, or share any of the following:

  • Personal information (name, email, account details)
  • Browsing history or visited URLs
  • Page content, screenshots, or PDF files you generate
  • Any data sent to external servers, third parties, or the developer

All processing happens entirely on your local device. No data ever leaves your browser.

3. Local Storage Usage

The Extension uses chrome.storage.local solely to store the version number of the last time you opened the extension popup. This single value is used exclusively to display a "NEW" badge on the extension icon when an update has been installed.

  • What is stored: Extension version string (e.g., "3.3.0")
  • Where it is stored: Locally in your browser only
  • Who can access it: Only the Extension itself, on your device
  • Is it shared or uploaded: Never

4. Permissions Explained

The Extension requests the following Chrome permissions:

  • activeTab — Access the current tab only when you click the extension icon or use the right-click menu. No background tab access.
  • scripting — Inject a capture script into the active tab to take a screenshot of the page.
  • contextMenus — Add a "Print Selection as PDF" option to the right-click menu when text is selected.
  • downloads — Save the generated PDF file to your local Downloads folder.
  • storage — Store the last-seen version string locally (see Section 3 above).

5. Third-Party Services

When you use the Gmail or Outlook compose features, the Extension opens a new browser tab to mail.google.com or triggers your system's mailto: handler. The Extension does not send any data to these services — it only pre-fills a compose window subject and body with the filename you just downloaded.

6. Children's Privacy

The Extension does not knowingly collect any information from anyone, including children under the age of 13.

7. Changes to This Policy

If this privacy policy changes, the updated policy will be posted at the same URL. Continued use of the Extension after changes constitutes acceptance of the new policy.

8. Contact

If you have questions about this privacy policy, please open an issue on the extension's support page in the Chrome Web Store, or contact the developer directly via the Chrome Web Store developer profile.

This privacy policy applies solely to the Chrome Extension "Text Print & Full Page Screenshot" and no other product or service.
